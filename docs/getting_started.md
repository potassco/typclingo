# Getting started

## Installation

=== "Pip"

    ```console
    pip install typclingo
    ```

=== "Development mode"

    ```console
    git clone https://github.com/potassco/typclingo.git/
    cd typclingo
    pip install -e .[all]
    ```

    !!! warning
        Use only for development purposes

## Usage

### Command line interface

Details about the command line usage can be found with:

```console
typclingo -h
```
